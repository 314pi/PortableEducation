Identifier l'accord

Le but de cet exercice est d'identifier l'accord joué.

Commencez l'exercice en pressant le bouton Nouveau. Solfege va jouer un accord et vous devrez l'identifier en cliquant sur l'un des boutons situés sous la portée vide.

Si vous avez deviné, le programme va afficher l'accord sur la portée avec le message "Très Bien". Vous pourrez alors presser le bouton Nouveau pour passer à la question suivante.

Si vous n'avez pas deviné, le message "Faux" sera affiché.
