The singanswer module

Here is a minimal lesson file:

header {
  module = singanswer
  version = "3.1.4"
  title = "Sing the root of the chord"
}
question {
    question_text = "Sing the root"
    music = chord("c' e' g'")
    answer = chord("c'")
}
question {
    question_text = "Sing the root"
    music = chord("a' c'' e''")
    answer = chord("a'")
}

Additional variables you can put in the header. Click on the link to get an explanation:
