Intervalle invertieren

You invert an interval when you move the lowest tone of an interval one octave higher or the highest tone one octave lower. The general name changes this way:

  • Eine Sekunde wird zur Septime.

  • Eine Terz wird zur Sexte.

  • Eine Quarte wird zur Quinte.

  • Eine Quinte wird zur Quarte.

  • Eine Sexte wird zur Terz.

  • Eine Septime wird zur Sekunde.

Die Intervallgruppe ändert sich wie folgt:

  • Vermindert wird zu übermäßig.

  • Dur wird Moll.

  • Perfekt bleibt perfekt.

  • Dur wird zu Moll

  • Übermäßig wird zu vermindert.

Nachstehend finden Sie zwei Beispiele. Eine große Terz wird umgekehrt und wird zur kleinen Sexte, und eine kleine Septime wird durch Umkehrung zur großen Sekunde.
