Редактор набора упражнений

The training set editor lets you create MIDI/WAV/MP3/OGG files of questions so that you can upload them to your pda, cell phone or MP3 player. A solution sheet will be generated for you to print out. Then you can let the MP3 player play the tracks in random order, and you can use the solution sheet to check if you recognized the music correctly.

Вы используете редактор набора упражнений, чтобы определить, какие упражнения генерировать. Вы можете сохранить Ваше определение в файле для использования позже. Каждый раз, когда Вы щёлкаете Экспорт, новый набор файлов генерируется в выбранном Вами каталоге. Вы должны вручную загрузить сгенерированные звуковые файлы в Ваше мобильное устройство.

Программа позволяет Вам генерировать вопросы из такого числа файлов уроков, какое пожелаете, но наиболее типичным использованием была бы генерация множества вопросов из одного или всего нескольких файлов.

Программы, используемые для конвертации между разными форматами файлов, определены на интерфейсной странице окна настроек. Пожалуйста, проверьте там параметры, если испытываете проблемы при конвертации MIDI-файлов в формат WAV, MP3 или OGG.

Объяснение заголовков таблицы

Число

Число вопросов для генерации из файла урока.

Повтор

Число повторов каждого вопроса.

Задержка

Насколько долгой будет задержка между вопросами. Измеряется в продолжительностях четвертных нот.
