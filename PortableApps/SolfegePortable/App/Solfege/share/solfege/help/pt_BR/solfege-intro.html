Bem vindo ao GNU Solfege

Solfege é um programa livre para treino de audição. O programa é parte do Projeto GNU. Verifique “Recursos na rede” para informações sobre listas de discussão e sobre onde obter a última versão do Solfege .

Uma das ideias deste programa é que você possa ampliá-lo sem ter que mexer no código fonte. Por exemplo, se você quiser praticar alguns acordes especiais, pode escrever um arquivo de lição por conta própria. Ler os arquivos é muito direto, e o jeito recomendado de se começar é copiar um dos arquivos existentes, e modificá-lo. Se você criar bons arquivos de lição, bem que poderia considerar compartilhá-los, enviando para a lista de discussão (em inglês), para que possamos adicioná-los à próxima versão deste programa.
