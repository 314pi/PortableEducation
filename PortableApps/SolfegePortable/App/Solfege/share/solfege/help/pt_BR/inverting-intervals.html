Invertendo intervalos

Você inverte um intervalo quando move a nota mais baixa do intervalo uma oitava acima, ou a nota mais alta uma oitava abaixo. O nome geral muda desta maneira:

  • Segunda se torna sétima.

  • Terça se torna sexta.

  • Quarta se torna quinta.

  • Quinta se torna quarta.

  • Sexta se torna terça.

  • Sétima se torna segunda.

O nome específico muda desta maneira:

  • Diminuta se torna aumentada.

  • Menor se torna maior.

  • Justa permanece justa.

  • Maior se torna menor.

  • Aumentada se torna diminuta.
